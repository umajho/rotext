# 注音

**注音**对应于 HTML 之中的 `ruby` + `rt` 元素。

参数：

- `1`：要被注音的内容；
- `2`：作为注音的内容。

目前在渲染时不考虑 `<rp />` 添加标签。

[[行内附加信息]]有效，作用于渲染后最外层元素之上。

## 示例

```example
[{#示例: 测试 | cè shì}]
···
<p><ruby>测试<rt>cè shì</rt></ruby></p>
```

```example
[! .foo][{#示例: 测试 | cè shì}]
···
<p><ruby class="foo">测试<rt>cè shì</rt></ruby></p>
```
