# 段落

**<ruby>段落<rt>Paragraph</rt></ruby>**&#x200B;是最基础的块级元素，对应<wbr />
于 HTML 之中的 `p` 元素。

- 开启：不能被解释为其他块级元素的行；
- [[s:块级阶段#块级元素槽位通用的终止界限|通用]]之外的终止界限：[[s:块级阶段#空行|空行]]之前；
- 槽位：其自身，填充[[s:行内阶段#行内序列|行内序列]]。

## 示例

```example
第一行
第二行
···
<p>第一行<br>第二行</p>
```

```example
第一段

第二段
···
<p>第一段</p>
<p>第二段</p>
```
